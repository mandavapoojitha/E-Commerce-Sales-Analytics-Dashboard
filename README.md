Project Overview
This project analyzes e-commerce sales data using Python, MySQL, and Google Looker Studio. 
The data is processed in Jupyter Notebook, stored in MySQL, and visualized through an interactive dashboard in Google Looker Studio.

 Objectives
- Analyze sales performance.
- Identify top-selling products and categories.
- Compare yearly and monthly sales trends.
- Evaluate weekend vs weekday sales.
- Generate business insights using SQL and Python.

Technologies Used
- Python
- Pandas
- MySQL
- Jupyter Notebook
- Google Looker Studio


Project Files
-`Merging and Analysis.ipynb` – Data cleaning, analysis, and visualization.
- `database.sql` – Database schema and SQL queries.
- `customer_detail.txt` – Customer dataset.
- `order_detail.txt` – Order dataset.
- `payment_detail.txt` – Payment dataset.
- `sku_detail.txt` – Product dataset.
- `Sales_Dashboard.pdf` – Dashboard report.
